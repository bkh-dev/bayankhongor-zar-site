# Баянхонгор аймгийн нэгдсэн зар

Баянхонгор аймгийн локал marketplace/demo сайт.

## Live Demo
https://bkh-dev.github.io/bayankhongor-zar-site/

## Төслийн тухай
Энэ төсөл нь HTML, CSS, JavaScript ашигласан local marketplace сайт юм.  
Хэрэглэгч зар үзэх, дэлгэрэнгүй хуудас руу орох, таалагдсан зар хадгалах, харьцуулах, inbox ашиглах, profile харах зэрэг боломжуудтай.

## Гол боломжууд
- Зар нэмэх, засах, устгах
- Дэлгэрэнгүй хуудас
- Favorites
- Compare
- Inbox
- Dark mode
- Profile
- Утасны дугаар + password нэвтрэлт
- LocalStorage ашигласан өгөгдөл хадгалалт
- Ангилал болон шүүлтүүр
- Үл хөдлөх ангиллын дэд төрөл
- Mobile responsive layout

## Ашигласан технологи
- HTML
- CSS
- JavaScript
- LocalStorage
- GitHub Pages

## Файлын бүтэц
- `index.html` — нүүр хуудас
- `script.js` — нүүр хуудасны logic
- `detail.html` — зарын дэлгэрэнгүй хуудас
- `detail.js` — detail page logic

## Анхаарах зүйл
Энэ төсөл нь demo зориулалттай.  
Одоогийн нэвтрэлт, өгөгдөл хадгалалт нь `localStorage` дээр тул production-ready backend auth биш.

## Цаашид сайжруулах зүйлс
- Backend болон database холбох
- Жинхэнэ authentication систем хийх
- Image upload storage ашиглах
- Admin dashboard сайжруулах
- Mobile UI-г илүү цэвэр болгох
